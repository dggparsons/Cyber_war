# Mega Challenge Outline

## Story Hook
"Unknown malware config discovered. Solve chained puzzles to neutralise it."

## Stage 1 – Cipher
- Solve a Vigenère cipher using key "DAWNSHIELD".
- Output reveals a Git repo URL.

## Stage 2 – Stego
- Download repo image, extract hidden base64.
- Decode to obtain coordinates / passphrase.

## Stage 3 – Malware Config
- Passphrase unlocks JSON config; identify kill-switch domain.

## Reward Tiers
1. **First solve**: +15 Influence, False Flag token, public shout-out.
2. **Second solve**: +10 Influence, Phone-a-Friend token.
3. **Third solve**: +5 Influence, +2 defense buff.
